# Demo plan

`sample-plan.pdf` is a **fully original, synthetic** floor plan — a tabloid sheet
with four enclosed rooms, room labels, a sheet number (`A-101`), and a drawn
scale note (`1/4" = 1'-0"`). No real project, no copyright. It exists so you can
try OpenTakeoff in ten seconds:

1. `npm run dev` in `../web`
2. Drag `sample-plan.pdf` onto the canvas
3. The scale auto-detects (`1/4" = 1'-0"`); pick a condition and click **One-Click
   Area** inside a room, or trace one with **Area**.

It's generated by [`make_sample_plan.py`](make_sample_plan.py) (pure Python, no
dependencies) so it's reproducible and auditable:

```bash
python3 make_sample_plan.py   # rewrites sample-plan.pdf
```

> Never commit a real client plan here — those are confidential and copyrighted.
> Drop your own PDFs in the app at runtime; they stay in your browser.
